# BytePixel
Open-source pixel display font
# Sciforge Pixel Font

BytePixel is a modular pixel-style open source display font.


## Features
- Uppercase A–Z
- Numbers 0–9
- Symbols
- Optimized for screens

## Use Cases
- UI dashboards
- Game HUDs
- Posters & titles
- Developer tools

## Installation
Download the font and install it like any standard .otf file.
